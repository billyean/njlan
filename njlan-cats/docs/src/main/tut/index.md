## Installation

njlan-cats is available on scala 2.11, 2.12, 2.13 and scalajs.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(
  "io.haiboyan.cats" %% "njlan-cats-core" % "0.0.1")
```
